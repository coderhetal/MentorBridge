# MentorBridge 🎯
### JEE Mentorship by IIT & NIT Seniors
 
> Connecting 11th & 12th students with IIT and NIT seniors for honest, one-on-one mentorship. Real talk from people who've been exactly where you are.
 
**Status:** Launched · Paused &nbsp;|&nbsp; Non-commercial · No fees · Voluntary
 
---
 
## 🧠 What is this?
 
MentorBridge is a peer mentorship platform built for JEE aspirants. Students fill a short Google Form describing their challenges, target, and availability. The team personally reviews every submission and matches each student with a verified IIT/NIT senior mentor for a one-on-one call.
 
No app downloads. No subscriptions. Just a real conversation.
 
Started by **VNIT Nagpur '25 alumni** who knew that what most JEE students are missing isn't content — it's *clarity from someone who's actually been there.* All mentors contribute voluntarily. There is no commercial interest behind this platform.
 
---
 
## ✨ Features
 
- **Mentor directory** — browsable cards with college, branch, availability, and help areas for each mentor
- **Live search + filter chips** — search by name, college, branch, day, or area; filter by mentoring focus (e.g. "Mock test analysis", "Study strategy")
- **Tag highlighting** — matching area tags are visually highlighted as you search
- **Empty state handling** — friendly message when no mentors match, with a fallback CTA
- **Google Form integration** — student registration links to a hosted Google Form (no backend required)
- **Scroll reveal animations** — Intersection Observer API for clean section entrances
- **Alumni vs Student badges** — mentors labelled clearly with status and year
- **Fully responsive** — mobile hamburger menu, stacked grid layouts, fluid typography
- **Zero dependencies** — no frameworks, no npm, no build step
